# SOC 2 Compliance Sandbox — Marineso

Automated SOC 2 Type II compliance auditing system.

## Controls Coverage
33 AICPA Common Criteria controls tracked automatically.

## Agents
- `scanner.py` — AWS / Okta / GitHub scanner
- `policy_agent.py` — AI-generated policy documents
- `github_agent.py` — GitHub compliance checks
- `hr_agent.py` — HR & training tracking
- `audit_runner.py` — One-button full audit
