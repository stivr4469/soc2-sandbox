# Security Policy

## Supported Versions
| Version | Supported |
|---------|-----------|
| main    | ✅ Active |

## Reporting a Vulnerability
**Do not open a public issue for security vulnerabilities.**

Report to: **security@marineso.com** — response within 5 business days.

## Disclosure Policy
- Coordinated Vulnerability Disclosure (CVD)
- Patch before public disclosure
- No legal action against good-faith researchers

## Scope
- `stivr4469/soc2-sandbox` code
- Evidence Tracker API
- All SOC 2 compliance agents
